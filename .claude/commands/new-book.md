description: Add a new book to the site by walking through the questionnaire one question at a time
---

Start the new-book questionnaire defined in `CLAUDE.md` (section: **Adding a New Book**).

Ask one question at a time, waiting for each answer before asking the next. After collecting all answers, build and insert the new `<article class="book-card">` block in `fiction.html` or `nonfiction.html` per the rules in `CLAUDE.md`, then show the user the snippet you inserted and ask if they want anything tweaked.
